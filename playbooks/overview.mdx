Playbooks are guided workflows that show you how to combine Qumis tools to accomplish real insurance tasks — from analyzing a prospect's coverage program to preparing a client renewal presentation. Each playbook walks you through a specific scenario step by step, so you can follow along with your own documents and get results immediately.

---

## What's in a playbook?

Every playbook includes a clear goal, the Qumis tools you'll use, step-by-step instructions, and tips for getting the best results. Think of them as recipes — follow the steps, swap in your own documents, and adapt the workflow to your situation.

## Playbooks for your role

<Tabs>
  <Tab title="Brokers & Producers">
    Start with the **Prospecting** playbooks to build pipeline, then use **Competitive Quoting** to compare options and **Presenting Quotes** to build client-ready deliverables. When it's time to bind, run a **Pre-Bind Review** to catch any gaps.

    Key playbooks: [Referral Mining](/playbooks/prospecting/referral-mining) → [Renewal Outreach](/playbooks/prospecting/renewal-outreach) → [Competitive Quoting](/playbooks/broking/competitive-quoting) → [Presenting Quotes](/playbooks/broking/client-presentation) → [Pre-Bind Review](/playbooks/binding/pre-bind-review)
  </Tab>
  <Tab title="Account Managers">
    Use **Mid-Term Coverage Check** to deliver proactive value between renewals, then prepare with **Renewal Preparation** and **Renewal Analysis** when it's time. **Portfolio Review** is ideal for annual program assessments.

    Key playbooks: [Mid-Term Coverage Check](/playbooks/servicing/mid-term-coverage-check) → [Portfolio Review](/playbooks/servicing/portfolio-review) → [Renewal Preparation](/playbooks/renewing/renewal-preparation) → [Renewal Analysis](/playbooks/renewing/renewal-analysis)
  </Tab>
  <Tab title="Claims Professionals">
    Start with **Coverage Determination** to build defensible coverage opinions, then use **Building Your Case** to support formal determinations with structured legal research.

    Key playbooks: [Coverage Determination](/platform/research/coverage-determination) → [Building Your Case](/platform/research/building-your-case)
  </Tab>
  <Tab title="Risk Managers">
    Use **Portfolio Review** for full program assessments, **Mid-Term Coverage Check** for ongoing monitoring, and **Pre-Bind Review** to verify new coverage before binding.

    Key playbooks: [Portfolio Review](/playbooks/servicing/portfolio-review) → [Mid-Term Coverage Check](/playbooks/servicing/mid-term-coverage-check) → [Pre-Bind Review](/playbooks/binding/pre-bind-review)
  </Tab>
</Tabs>
